### 📈 Objetivo

Esse template foi feito com NextJS, Typescript, EsLint, Prettier, Docker e Husky.

<br />
<hr>
 
### 📝 Como utilizar

```bash
# Install dependencies
$ yarn

# Run the project in development mode
$ yarn dev

# Build the project
$ yarn build

# Run the project in production mode
$ yarn start
```

<br />
